==标签和分支==：

> * 标签：是一个静态的概念，打上标签之后，这些文件始终是这个状态
>   * 使用场景： 当一个项目开发到一个差不多的时候，打上一个标签，表示一个版本
>
> * 分支：动态的，在这个分支可以继续开发代码，不断增加新的内容
